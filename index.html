Lesh NGCA Tab 隱私權政策

最後更新：2026年4月18日

收集的資料

本擴充功能僅在使用者授權時存取裝置的 GPS 位置資訊，用於顯示當地即時天氣。位置資料不會被儲存、傳送至任何伺服器或用於其他用途。

第三方服務

本擴充功能連接以下外部服務以提供功能：

  • wttr.in / open-meteo.com — 天氣資料
  • Google Calendar API — 行事曆事件（需使用者授權）
  • suggestqueries.google.com — 搜尋自動完成
  • SomaFM — 背景音樂串流
  • Firebase Realtime Database — 即時資料同步

資料不會被販售或分享

本擴充功能不會收集、販售或將任何使用者資料轉給第三方。所有資料僅用於提供擴充功能本身的功能。
